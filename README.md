# מערכת מתכונים לטאבלט

אפליקציית HTML פשוטה לניהול והצגת מתכונים לעובדי ייצור.

## תכונות

- ✅ בחירת מתכון וכמות (בכפולות של 10 ק"ג)
- ✅ שלב קיט - חישוב אוטומטי של כמויות רכיבים
- ✅ שלב תנור - הוראות אפייה (כמות מים משתנה לפי הכמות)
- ✅ ממשק ניהול להוספה/עריכה/מחיקה של מתכונים
- ✅ עיצוב מותאם לטאבלט
- ✅ שמירה ב-Google Sheets דרך Apps Script

## שימוש

1. פתח את `index.html` בדפדפן
2. לעובדים: בחר מתכון → בחר כמות → עקוב אחר ההוראות
3. למנהלים: פתח `admin.html` להוספת/עריכת מתכונים

## הגדרה

1. צור Google Sheet בשם "מתכונים"
2. צור Google Apps Script (ראה `GOOGLE_SHEETS_SETUP.md`)
3. עדכן את ה-URL ב-`app.js`

## קבצים

- `index.html` - דף ראשי (בחירת מתכון)
- `kit.html` - שלב קיט
- `oven.html` - שלב תנור
- `admin.html` - ניהול מתכונים
- `app.js` - לוגיקה ראשית
- `admin.js` - לוגיקת ניהול
- `styles.css` - עיצוב

## דפלוי

ראה `DEPLOY.md` להוראות פרסום ל-GitHub Pages או Netlify.
