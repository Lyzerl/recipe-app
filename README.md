# מערכת מתכונים לטאבלט

אפליקציית HTML פשוטה לניהול והצגת מתכונים לעובדי ייצור.

## תכונות

- ✅ **קובץ HTML אחד** - הכל בקובץ בודד, קל לשיתוף והתקנה
- ✅ בחירת מתכון וכמות (בכפולות של 10 ק"ג)
- ✅ שלב קיט - חישוב אוטומטי של כמויות רכיבים
- ✅ חלוקה לתבניות - גסטרונום / סיפט
- ✅ שלב תנור - הוראות אפייה (גסטרונום / סיפט)
- ✅ ממשק ניהול להוספה/עריכה/מחיקה של מתכונים
- ✅ עיצוב מותאם לטאבלט ומובייל (רספונסיבי)
- ✅ שמירה ב-Google Sheets דרך Apps Script
- ✅ תמיכה מלאה בגסטרונום וסיפט עם כל הפרמטרים

## שימוש

1. פתח את `index.html` בדפדפן (או בטלפון/טאבלט)
2. **לעובדים**: בחר מתכון → בחר פעולה (קיטים/חלוקה/תנור) → עקוב אחר ההוראות
3. **למנהלים**: לחץ "הוסף מתכון" או "ערוך מתכונים"

## הגדרה

### שלב 1: הגדרת Google Apps Script

1. צור Google Sheet חדש
2. לך ל-[Google Apps Script](https://script.google.com)
3. צור פרויקט חדש
4. העתק את כל הקוד מ-`APPS_SCRIPT_CODE.js`
5. שמור (Ctrl+S)
6. לחץ **פרסום** → **פרוס כאפליקציית אינטרנט**
7. בחר:
   - **הרצה כ**: אני (החשבון שלך)
   - **מי יכול לגשת**: **כל אחד**
8. לחץ **פרוס** והעתק את ה-URL

### שלב 2: עדכון ה-URL

1. פתח את `index.html`
2. מצא את השורה: `const APPS_SCRIPT_URL = '...'`
3. החלף את ה-URL ב-URL החדש מ-Apps Script
4. שמור

## קבצים

### קבצים חיוניים:
- ✅ **`index.html`** - הקובץ היחיד שצריך! (מכיל הכל: HTML, CSS, JavaScript)

### קבצים מומלצים:
- 📝 `APPS_SCRIPT_CODE.js` - הקוד ל-Google Apps Script (לצורך עדכון עתידי)

### קבצים שלא נדרשים:
- ❌ `app.js`, `main.js`, `styles.css` - כבר משולבים ב-index.html
- ❌ כל קבצי ה-MD - רק הוראות, לא נדרשים

## דפלוי

### אפשרות 1: GitHub Pages
1. העלה את `index.html` ל-GitHub
2. לך ל-Settings → Pages
3. בחר Branch: main, Folder: / (root)
4. שמור
5. ה-URL יהיה: `https://[username].github.io/[repository-name]/`

### אפשרות 2: שימוש ישיר
פשוט פתח את `index.html` בדפדפן - זה יעבוד!

## תמיכה

אם יש בעיות, בדוק:
- שה-URL ב-Apps Script נכון
- שהפריסה ב-Apps Script היא "כל אחד"
- את הקונסול בדפדפן (F12) לשגיאות
