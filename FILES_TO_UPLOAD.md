# קבצים להעלאה ל-GitHub

העלה את הקבצים הבאים ל-GitHub Repository:

## קבצים חובה (הכרחיים):

1. ✅ `index.html` - דף ראשי
2. ✅ `kit.html` - שלב קיט
3. ✅ `oven.html` - שלב תנור
4. ✅ `admin.html` - ניהול מתכונים
5. ✅ `app.js` - לוגיקה ראשית
6. ✅ `admin.js` - לוגיקת ניהול
7. ✅ `styles.css` - עיצוב
8. ✅ `README.md` - תיאור הפרויקט
9. ✅ `.gitignore` - קובץ Git

## קבצים אופציונליים (תיעוד):

- `GOOGLE_SHEETS_SETUP.md` - הוראות הגדרת Google Sheets
- `DEPLOY.md` - הוראות דפלוי
- `GITHUB_SETUP.md` - הוראות GitHub
- `FIX_PERMISSIONS.md` - פתרון בעיות הרשאות
- `QUICK_FIX.md` - תיקון מהיר

## קבצים לא להעלות:

- ❌ `test-url.html` - קובץ בדיקה מקומי
- ❌ `recipis.gsheet` - קובץ Google Sheets מקומי
- ❌ כל קובץ עם `.md` אם אתה לא רוצה תיעוד

---

## הוראות מהירות:

1. לך ל-GitHub → צור Repository חדש (Public)
2. לחץ "uploading an existing file"
3. גרור את כל הקבצים מהרשימה למעלה
4. Commit changes
5. Settings → Pages → בחר main → Save
6. תקבל URL!
